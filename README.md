# Установка!:

* apt update && apt upgrade
* pkg install git
* git clone https://github.com/vladwdw/bomber
* sh bomber/install.sh
* Готово! 

# Обновление бомбера!
* rm -rf bomber
* git clone https://github.com/vladwdw/bomber
* cd bomber
* python3 bomber.py
